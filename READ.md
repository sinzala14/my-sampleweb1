# My Sample Web 1

This project is a simple static website created for testing and preview purposes using GitHub.

## 🌐 Live Preview
After enabling GitHub Pages, the website will be available at:

https://sinzala14.github.io/my-sampleweb1/

## 🛠 Technologies Used
- HTML
- CSS
- JavaScript

## 📁 Files Included
- index.html – Main webpage
- CSS files – Styling
- JavaScript files – Interactivity
- Assets (images, icons, etc.)

## 🚀 How to View
- Open `index.html` in a browser  
OR  
- View it online using GitHub Pages

## ⚙ Deployment Steps (GitHub Pages)
1. Go to **Settings**
2. Click **Pages**
3. Select:
   - Branch: `main`
   - Folder: `/root`
4. Save and wait a moment
